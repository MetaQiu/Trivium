# Trivium

Multi-agent collaborative paper writing system. Orchestrates Claude Code, Codex CLI, and Gemini CLI to independently draft, cross-review, argue, and reach consensus on academic papers.

## Structure

```
scripts/        # Main orchestrator (WIP)
templates/      # Prompt templates (WIP)
deps/           # Bridge scripts from GuDaStudio
```

## Dependencies

- Claude Code CLI
- OpenAI Codex CLI
- Google Gemini CLI
- Python 3.10+
